### Golang

Para executar a resolução deste pacote basta instalar a última versão do [Go](https://go.dev/dl/), depois entrar no diretorio:

```shell
cd go/vivi3008/"nomeDaPastadoChallenge"
```

E rodar no seu terminal:

```shell
go run main.go
```
